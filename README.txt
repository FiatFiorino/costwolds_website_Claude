COSTWOLDS LLC WEBSITE - INSTALLATION GUIDE
==========================================

CONTENTS OF THIS PACKAGE:
-------------------------
1. index.html - Your main website file
2. styles.css - The stylesheet with all design elements
3. images/ - Folder containing your brand images
   - logo-background.jpg
   - hero-background.jpg

INSTALLATION INSTRUCTIONS:
--------------------------

OPTION 1: Simple Hosting (Netlify, Vercel, GitHub Pages)
1. Create an account on Netlify (netlify.com) or similar service
2. Drag and drop this entire folder onto their dashboard
3. Your site will be live in seconds!
4. You can add a custom domain later in their settings

OPTION 2: Traditional Web Hosting (GoDaddy, Bluehost, etc.)
1. Log into your web hosting control panel (cPanel)
2. Navigate to File Manager
3. Go to the public_html folder (or www folder)
4. Upload all files:
   - index.html
   - styles.css
   - images folder (with both images inside)
5. Your site will be live at your domain

OPTION 3: WordPress (if you use WordPress)
1. Install "Simple Custom CSS and JS" plugin
2. Create a new page in WordPress
3. Set the page template to "Blank" or "Full Width"
4. Copy the content from index.html (everything inside <body> tags)
5. Paste into the page editor
6. Add styles.css content to the Custom CSS section
7. Upload images to WordPress media library
8. Update image paths in the HTML if needed

CUSTOMIZATION:
--------------
To change colors, text, or layout:
1. Open styles.css in any text editor (Notepad, TextEdit, VS Code)
2. Colors are defined at the top of each section
3. Main brand colors: #c89666, #d4a574, #b8875a

To update text content:
1. Open index.html in any text editor
2. Find the section you want to change
3. Edit the text between HTML tags
4. Save and re-upload
